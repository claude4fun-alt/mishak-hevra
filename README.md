# 🎮 משחק חברה - מערכת משחק תחנות QR רב-חברתית

מערכת משחק תחנות אינטראקטיבי לכמה חברות במקביל, עם בידוד מלא בין החברות:
10 תחנות QR לכל חברה, שאלות טקסט/תמונה, ניקוד ובונוס מהירות, לוח תוצאות חי,
ייצוא/ייבוא של סטים של שאלות, וגיבוי ל-GitHub.

## הפעלה
```
npm install
npm start
```

## משתני סביבה (לגיבוי ל-GitHub - אופציונלי אך מומלץ)
- `GH_TOKEN` - fine-grained token עם הרשאת Contents: Read and write
- `GH_OWNER` - שם המשתמש בגיטהאב
- `GH_REPO` - שם ה-repo לנתונים (פרטי, נפרד מה-repo של הקוד)

## תפקידים
- מנהל ראשי: `amirco` / `Havefun360` - יוצר ומנהל חברות
- מנהל חברה: נכנס עם שם החברה + סיסמה שקיבל מהמנהל הראשי

## קישורים
- `/dashboard` - ניהול (ראשי או חברה)
- `/c/:id` - עמוד ראשי של חברה (יעד ה-QR הראשי)
- `/c/:id/board` - לוח תוצאות חי
- `/c/:id/station/1..10` - תחנות
